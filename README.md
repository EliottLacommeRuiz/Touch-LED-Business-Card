This project is all about creating something with no step by step tutorial.

I want to design a buisness card sized PCB, that has a capacitive touchpad on it and can have my gestures be displayed on an LED matrix. I also want to include an edge PCB usb C connector since it looks cool. 

This is a very exciting project that requires a lot of design work for the touchpad and careful firmware managing for the LED matrix. 

I plan on doing this project throughout my spring quarter of my junior year in college and potentially into the summer.
